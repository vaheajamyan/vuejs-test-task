# vue-test

> My tiptop Nuxt.js project

## Build Setup

```bash
# install dependencies
$ cd vue-test/
$ npm install

$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
